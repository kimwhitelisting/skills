# 테마팩토리

## 문서 정보

- **이름**: `테마팩토리`
- **설명**: 테마를 사용하여 아티팩트 스타일을 지정하기 위한 툴킷입니다. 이러한 아티팩트는 슬라이드, 문서, 보고, HTML 랜딩 페이지 등이 될 수 있습니다. 생성 중인
          모든 아티팩트에 적용할 수 있거나 즉시 새 테마를 생성할 수 있는 색상/글꼴이 포함된 사전 설정된 테마 10개가 있습니다.
- **라이선스**: -


이 기술은 엄선된 전문 글꼴 및 색상 테마 테마 컬렉션을 제공하며 각 테마에는 신중하게 선택된 색상 팔레트와 글꼴 조합이 포함되어 있습니다. 테마를 선택하면 모든 아티팩트에
적용할 수 있습니다.

## 목적

프레젠테이션 슬라이드 데크에 일관되고 전문적인 스타일을 적용하려면 이 기술을 사용하십시오. 각 테마에는 다음이 포함됩니다.
- 16진수 코드가 포함된 응집력 있는 색상 팔레트
- 헤더와 본문 텍스트에 대한 보완적인 글꼴 조합
- 다양한 상황과 대상에 적합한 뚜렷한 시각적 아이덴티티

## 사용 지침

슬라이드 데크나 기타 아티팩트에 스타일을 적용하려면 다음을 수행하십시오.

1. **테마 쇼케이스 표시**: 사용자가 사용 가능한 모든 테마를 시각적으로 볼 수 있도록 `theme-showcase.pdf` 파일을 표시합니다. 이를 수정하지
   마십시오. 보기 위해 파일을 보여주기만 하면 됩니다.
2. **선택을 요청**: 덱에 어떤 테마를 적용할지 물어보세요.
3. **선택 대기**: 선택한 테마에 대한 명시적인 확인을 받습니다.
4. **테마 적용**: 테마를 선택하고 나면 선택한 테마의 색상과 글꼴을 덱/아티팩트에 적용합니다.

## 사용 가능한 테마

다음 10가지 테마를 사용할 수 있으며 각각 `theme-showcase.pdf`으로 표시됩니다.

1. **Ocean Depths** - 전문적이고 차분한 해양 테마
2. **Sunset Boulevard** - 따뜻하고 생생한 일몰 색상
3. **숲 캐노피** - 자연스럽고 접지된 흙 톤
4. **모던 미니멀리스트** - 깔끔하고 현대적인 그레이스케일
5. **골든 아워** - 풍부하고 따뜻한 가을 팔레트
6. **Arctic Frost** - 시원하고 산뜻한 겨울에서 영감을 받은 테마
7. **데저트 로즈** - 부드럽고 세련된 더스티 톤
8. **기술 혁신** - 대담하고 현대적인 기술 미학
9. **식물원** - 신선하고 유기농 정원 색상
10. **Midnight Galaxy** - 드라마틱하고 우주적인 깊은 톤

## 테마 세부정보

각 테마는 다음을 포함한 전체 사양과 함께 `themes/` 디렉토리에 정의됩니다.
- 16진수 코드가 포함된 응집력 있는 색상 팔레트
- 헤더와 본문 텍스트에 대한 보완적인 글꼴 조합
- 다양한 상황과 대상에 적합한 뚜렷한 시각적 아이덴티티

## 지원 절차

선호하는 테마를 선택한 후:
1. `themes/` 디렉터리에서 해당 테마 파일을 읽습니다.
2. 지정된 색상과 글꼴을 데크 전체에 일관되게 적용합니다.
3. 적절한 대비와 가독성 보장
4. 모든 슬라이드에서 테마의 시각적 정체성을 유지합니다.

## 나만의 테마 만들기
기존 테마가 아티팩트에 작동하지 않는 경우를 처리하려면 사용자 정의 테마를 생성하십시오. 제공된 입력을 기반으로 위와 유사한 새 테마를 생성합니다. 글꼴/색상 조합이 무엇을
나타내는지 설명하는 비슷한 이름을 테마에 지정합니다. 제공된 기본 설명을 사용하여 적절한 색상/글꼴을 선택하세요. 테마를 생성한 후 검토 및 확인을 위해 표시하세요. 그런 다음
위에서 설명한 대로 테마를 적용합니다.
