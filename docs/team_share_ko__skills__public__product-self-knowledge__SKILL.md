# 제품 자체 지식

## 문서 정보

- **이름**: `제품 자체 지식`
- **설명**: "귀하의 응답에 Anthropic 제품에 대한 특정 사실이 포함될 때마다 이 기술을 중지하고 참조하십시오. 포함: Claude 코드(설치 방법, Node.js
          요구 사항, 플랫폼/OS 지원, MCP 서버 통합, 구성), Claude API(함수 호출/도구 사용, 일괄 처리, SDK 사용, 속도 제한, 가격 책정,
          모델, 스트리밍) 및 Claude.ai(Pro 대 Team 대 Enterprise 계획, 기능 제한). Anthropic SDK를 사용하는 코딩 작업, 콘텐츠
          생성 언급에 대해서도 이 기술을 트리거합니다. Claude 기능이나 가격 또는 LLM 제공업체 비교 Anthropic 제품 세부 정보를 메모리에 의존할 경우
          언제든지 여기에서 확인하세요. 교육 데이터가 오래되었거나 잘못되었을 수 있습니다.

## 핵심 원칙

1. **추측보다 정확성** - 확실하지 않은 경우 공식 문서를 확인하세요.
2. **제품 구별** - Claude.ai, Claude Code, Claude API는 별도의 제품입니다.
3. **모든 것을 소스로 제공** - 항상 공식 문서 URL을 포함하세요.
4. **올바른 리소스 우선** - 각 제품에 대해 올바른 문서를 사용합니다(아래 라우팅 참조).

## 질문 라우팅

### Claude API 또는 Claude Code 질문이 있으신가요?

→ **문서 맵을 먼저 확인한 후** 특정 페이지로 이동하세요.

- **Claude API 및 일반:** https://docs.claude.com/en/docs_site_map.md
- **클로드 코드:** https://docs.anthropic.com/en/docs/claude-code/claude_code_docs_map.md

### Claude.ai 질문이 있으신가요?

→ **지원 페이지 찾아보기:**

- **Claude.ai 도움말 센터:** https://support.claude.com

## 응답 작업 흐름

1. **제품 식별** - API, Claude Code 또는 Claude.ai?
2. **올바른 리소스 사용** - API/코드용 문서 맵, Claude.ai 지원 페이지
3. **세부정보 확인** - 특정 문서 페이지로 이동합니다.
4. **답변 제공** - 소스 링크를 포함하고 어떤 제품인지 명시하세요.
5. **불확실한 경우** - 사용자를 관련 문서로 안내합니다. "최신 정보는 [URL]을 참조하세요."
